# Vue 3 简单的模板

整合了：Vue 3 + TypeScript  + vuex + vue-router + axios 

## 项目目录
```
src
├──assets         资源目录
├──components     项目组件目录
├──router         
│  ├──index.ts    路由配置
│  └──login.ts    路由的登入认证
├──store
│  └──index.ts    Vuex的配置
├──util
│  └──request.ts  axios的封装
├──view
│  ├──demo.ts     demo界面
│  └──Login.ts    简单的登入见面
├──App.vue        App
└──main.ts        主程序入口
```
